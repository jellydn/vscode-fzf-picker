## Key Locations

| Location               | Significance                                                                 |
| ---------------------- | ---------------------------------------------------------------------------- |
| `src/extension.ts:598` | Largest production file вЂ” terminal management, command dispatch, config init |
| `src/commands.ts`      | CLI entry that runs in terminal вЂ” all file opening logic                     |
| `src/commands/*.ts`    | 4 command modules + 1 UI module                                              |
| `src/config.ts`        | Shared mutable state (`CFG`) вЂ” used by both extension.ts and type-filter.ts  |

## Naming Conventions

- **Files**: kebab-case (`find-files.ts`, `live-grep.ts`)
- **Exports**: camelCase functions (`findFiles`, `liveGrep`, `openFiles`)
- **Types**: PascalCase (`Config`, `CustomTask`, `RuntimeType`)
- **Constants**: UPPER_SNAKE_CASE (`TERMINAL_NAME`, `DEBUG`)
- **VS Code config keys**: dot-notation `fzf-picker.general.batTheme`
- **Commands**: prefix `fzf-picker.` (e.g., `fzf-picker.findFiles`)

## Generated vs Hand-Written

- `src/generated/meta.ts` вЂ” auto-generated by `vscode-ext-gen` package, do not edit
- Everything else вЂ” hand-written
